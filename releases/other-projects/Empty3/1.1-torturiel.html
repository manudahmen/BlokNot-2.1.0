Introduction

Bonjour et bienvenue

Empty3 est destiné à la création d'images de synthèse en perspective 3D.

Empty3 est une bibliothèque de classes. Il y a néamoins un point d'entrée applicatif qui fait qu'elle n'est pas véritablement une bibliothèque de classe. En fait, elle devrait être séparée en 2 parties: la partie classe, et la partie "cliente" ou "créative"

La partie cliente est représentée par un ensemble de classe dont une classe implémentée par l'utilisateur de la bibliothèque dérivée de la classe TestObjet peut être pourvu d'une méthode main() par exemple pour générer, qui des modèles STL, qui une séquence d'images, qui une vidéo

Où trouver Empty3?

Sur le site de développement Github.com

Tout d'abord le langage: Java. Empty3 est développé dans ce langage. C'est dans ce langage que nous allons apprendre à créer nos premières vidéos.

La déclaration "package" : vous pouvez placer vos classes où bon vous semble dans la hiérarchie des classes de votre projet.

Plusieurs classes dans le package info.emptycanvas.library.objects, tribase qui contient les surfaces paramétriques prédéfinies, testing qui contient la classe TestObjetStub, classe canevas utilisée pour la génération de vidéos..

La lecture du script est assez facile.

  • Les méthodes surchargées, testScene() qui contient les instructions de modification de scène.
  • finit() initialisation de chaque frame.
  • ginit() initialisation "globale", appelée avant le premier appel à finit() puis testObjet()
  • La méthode main() qui est le point d'entrée de l'"application"
  • Les méthodes de TestObjet telles que scene() qui renvoie l'objet (de type Scene) contenant les éléments à dessiner (primitives)
